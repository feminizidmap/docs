Mapper deployment

Deployment setup

Heroku

You can run mapper on Heroku’s ‘hobby’ infrastructure. You need two heroku apps, one for the backend and one for the frontend. Attach a postgres and redis addon to the backend app. Add the required environment variables to each and you’re done. See the github/deploy workflow for reference.
